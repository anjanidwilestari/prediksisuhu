Tentang Website

Deskripsi:

Website ini merupakan bagian dari penelitian skripsi dengan judul "Prediksi Suhu Udara Menggunakan Backpropagation dengan Memanfaatkan Seleksi Fitur Berbasis Algoritma Genetika". Website ini dirancang untuk memprediksi suhu udara berdasarkan jumlah awan. Terdapat 3 fitur yaitu memasukkan data, melihat hasil prediksi, dan mengunduh hasil prediksi.

Metode:

Metode yang digunakan dalam prediksi suhu udara adalah algoritma backpropagation dengan memanfaatkan seleksi fitur berbasis algoritma genetika.

Cara Kerja:

Nilai yang telah dimasukkan oleh pengguna akan diubah menjadi nilai Nh Label. Nilai Nh Label digunakan sebagai input untuk model prediksi. Sistem memproses nilai Nh Label hingga menghasilkan prediksi suhu udara.

Glosarium

Nh:

Jumlah semua awan (%) CL yang ada atau, jika tidak ada awan CL, jumlah semua awan CM yang ada.

Nh Label:

Nh Label adalah hasil konversi dari nilai Nh ke suatu label

CL:

Awan dari genera Stratocumulus, Stratus, Cumulus, dan Cumulonimbus.

CM:

Awan dari marga Altocumulus, Altostratus dan Nimbostratus.

T:

Suhu udara (°C) pada ketinggian 2 meter di atas permukaan bumi.

Mengukur Tutupan Awan

Untuk mulai mengamati awan di suatu area, pertama-tama perkirakan total tutupan awan di seluruh langit. Pastikan juga untuk memperkirakan tutupan awan untuk setiap tingkat ketinggian; misalnya, rendah, sedang, atau tinggi. Anda dapat menggunakan gambar berikut untuk menentukan label yang sesuai dengan kondisi langit Anda.

Sumber: The Globe Program

Tips pengukuran:

Ketika mengamati langit di atas kepala, pastikan untuk tidak memasukkan horison. Untuk membuat pengukuran tutupan awan yang akurat:

  • Amati di atas sudut 14 derajat dengan horison
  • Atau:
    1. Rentangkan kedua tangan Anda membentuk huruf "V"
    2. Pastikan tangan Anda sejajar dengan bagian atas kepala Anda
    3. Ambil pengamatan tutupan awan di area antara kedua tangan Anda
  • Atau:
    1. Rentangkan tangan Anda dengan mengepal
    2. Pastikan bagian bawah tangan Anda sejajar dengan horison
    3. Tumpukkan tangan kedua di atas tangan pertama
    4. Ambil pengamatan tutupan awan di atas tangan kedua Anda

Panduan Penggunaan

Input Manual:

  1. Masukkan nilai Persentase Jumlah Semua Awan (%).
  2. Masukkan nilai Target Suhu (°C).
  3. Klik tombol "Prediksi" untuk mendapatkan hasil prediksi suhu.

Unggah File Excel:

  1. Siapkan file Excel dengan header kolom [Nh] dan [T].
  2. Pastikan tidak ada nilai NaN dalam file Excel.
  3. Unggah file dengan klik "Choose File" dan pilih file Excel yang sesuai.
  4. Klik tombol "Unggah" untuk memproses file dan mendapatkan hasil prediksi.

Anda dapat mengunduh file template dengan klik "Unduh format".

Tampilan Hasil:

Hasil prediksi akan ditampilkan dalam tabel dengan kolom Jumlah Awan, Nh Label, dan Prediksi Suhu. Anda dapat mengunduh hasil prediksi dalam format Excel dengan klik tombol "Unduh Prediksi".
